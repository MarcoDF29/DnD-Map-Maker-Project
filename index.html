Controles & Atajos de Teclado

Acción Control del Ratón / Teclado
Moverse por el mapa (Pan) Mantén pulsado Espacio + arrastrar ratón, o usa el botón derecho + arrastrar.
Hacer Zoom Usa la Rueda del ratón o los botones ➕ / ➖ en la barra superior.
Pintar continuo (Terreno/Niebla) Haz clic izquierdo y arrastra por las casillas.
Colocar un Objeto/Ficha Selecciona un objeto del panel, haz clic izquierdo en el mapa para colocarlo.
Seleccionar / Mover un Objeto Con la herramienta Objetos activa, haz clic izquierdo en un objeto del lienzo para seleccionarlo y arrástralo para moverlo.
Rotar Objeto Seleccionado Pulsa la tecla R para girarlo 15° en sentido horario, o usa la barra en el panel.
Eliminar Objeto Seleccionado Pulsa las teclas Supr / Retroceso (Backspace).
Duplicar Objeto Seleccionado Pulsa Ctrl + D o usa el botón "Duplicar".
Deshacer / Rehacer acción Usa Ctrl+Z para deshacer y Ctrl+Y para rehacer.
Movimiento libre (Sin rejilla) Mantén pulsado Alt mientras arrastras un objeto para posicionarlo con precisión.
